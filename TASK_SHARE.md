## 요청 사항 / 알려진 이슈 

- 회원 가입시 일반회원(탈렌트)과 기업회원(에이젼시)의 입력 속성값 확인 필요 (박승준 / 김정근)
- 일반회원 (탈렌트)의 입력 속성값 확인 필요 (박승준 / 김정근)
- 영상에 대한 속성값 정의. 년대, 역활, 등? (박승준 / 김정근)
- 연기자 샘플 자료 전달 (회원 가입 속성에 맞고 영상 자료 포함, 사진 등)
- 박승준 대표님의 접속 ID, 이메일, 패스워드 전달 (김정근)
